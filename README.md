AI Text Generator

📌 Overview
A simple AI text generation project using Hugging Face Transformers.  
It takes a user prompt and generates text output.

⚙️ Tech Stack
- Python
- Hugging Face Transformers (GPT‑2 / GPT‑Neo)
- Anaconda / Jupyter Notebook

 🚀 Features
- Generate text from any prompt
- Easy to run in Anaconda environment
- Extendable to stronger models

📂 Project Structure
- `ai_text.py` → Main script
- `requirements.txt` → Dependencies
- `README.md` → Documentation
